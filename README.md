# Student Management System

This project is a simple Java-based Student Management System that demonstrates core Object-Oriented Programming (OOP) concepts such as inheritance, interfaces, and encapsulation.

## Features
- Stores student details like name, age, ID, roll number, and marks
- Calculates student grade based on marks using an interface
- Displays complete student information

## Concepts Used
- Inheritance (Person → Student)
- Interface (GradeCalculation)
- Method Overriding
- Encapsulation

## Output
The program creates a student object and displays their details along with the calculated grade.

## Language Used
- Java
